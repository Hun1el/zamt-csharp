Практическая работа № 18
"Прототипирование интерфейса пользователя"

Цель работы:
Закрепить полученные навыки по работе с технологией WinForm и принципом построения интерфейса пользователя в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio создать проект и написать линейные программы, использующие технологию WinForm.
  2. ВНИМАНИЕ!!! Во всех заданиях варианта начальные значения должны вводиться с формы, используя соответствующие контроллы. Выходные значения (ответы) во всех заданиях варианта необходимо выводить в другие контролы на форму
  3. Минимальный размер массива должен быть не менее 20 элементов. Минимальный размер строк(и) (массив символов) - не менее 30 символов.
  4. Интерфейс программы должен быть четко продуман, control'ы Visual Studio (TextBox/Label/RadioButton/CheckBox/MenuStrip/...) при построении интерфейса должны распологаться "не хаотично" и не как попало, а представлять полностью законченный дизайн формы и приложения (ПР) в целом. Для прототипирования интерфейса рекомендуется использовать средства MS Visio или бумажный элемент прототипа.
  5. Сдать проект, показав полностью готовый интерфейс программы с выполненными заданиями исходя из Вашего варианта
  6. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  7. ВНИМАНИЕ!!! В заданиях, где требуется работа с массивом(и) - обязательно вводить размер массива(ов) и значения элементов массива(ов) с формы.
  8. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...).
  9. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  10. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

2. Дано вещественное число A и целое число N (N > 0). Вывести значение результата A в степени N: AN = A*A*...*A (число A перемножается N раз).
3. Вводится строка-предложение на английском языке. Длина строки может быть разной. Определить и вывести слово, содержащего наибольшее число символов 'd'.
4. Вводится строка, состоящая из слов разделенных точками. Длина строки может быть разной. Сформировать и вывести подстроку, расположенную между второй и пятой точками исходной строки. Если в строке менее пяти точек, то вывести всю исходную строку.
ВАРИАНТ № А38/Б8
1. Дано четырехзначное целое ненулевое положительное число N (N>0). Проверить истинность высказывания: "Все цифры данного числа различны".
2. Даны пять целых ненулевых положительных чисел. Найти сумму двух наименьших чисел.
3. Вводится строка, изображающая целочисленное арифметическое выражение вида «цифра_цифра_цифра_цифра», где на месте знака операции «_» находится символ «+» или «-», а на месте "цифра" находится одна из цифр (от 1 до 9). Например, «4+7-2+5». Вывести значение данного выражения (как целое число).
4. Вводится строка, изображающая целое число. Длина строки может быть разной. Вывести на экран произведение всех четных цифр этого числа.
ВАРИАНТ № А39/Б7
1. Проверить истинность высказывания: "Сумма цифр данного положительного трехзначного числа является четным числом".
2. Дано целое число N (N > 0). Вывести произведение 1*2*3*...*N. Чтобы избежать целочисленного переполнения, вычислять это произведение с помощью вещественной переменной и выводить его на экран как вещественное число. Пример для числа 5 - 1*2*3*4*5=120.
3. Вводится строка, состоящая из слов, разделенных подчеркиваниями (одним или несколькими). Длина строки может быть разной. Определить и вывести количество слов, которые содержат ровно одну букву 'h'.
4. Вводится строка, состоящая из слов, разделенных подчеркиваниями (одним или несколькими). Длина строки может быть разной. Определить и вывести количество слов, у которых второй и последний символ одинаковые.
ВАРИАНТ № А40/Б6
1. Проверить истинность высказывания: "Квадратное уравнение A·x2 + B·x + C = 0 с данными коэффициентами A (A ≠ 0), B, C имеет ровно два вещественных корня".
2. Ввести пять различных ненулевых целых чисел. Найти произведение двух наибольших чисел.
3. Вводится строка, состоящая из слов, разделенных подчеркиваниями. Вывести строку, содержащую эти же слова, но разделенные одним символом '.' (точка). Найти и вывести самое короткое слово из этой строки.
4. Вводится строка, состоящая из слов разделенных точками. Длина строки может быть разной. Сформировать и вывести подстроку, расположенную между второй и пятой точками исходной строки. Если в строке менее пяти точек, то вывести всю исходную строку.
ВАРИАНТ № А41/Б5
1. Проверить истинность высказывания: "Данное целое положительное число является четным трехзначным числом".
2. Даны шесть целых ненулевых положительных чисел. Найти сумму трех наименьших чисел.
3. Вводится строка, содержащая латинские буквы и круглые скобки. Длина строки может быть разной. Если скобки расставлены правильно (т. е. каждой открывающей скобки соответствует одна закрывающая скобка), то вывести число 0. В противном случае вывести или номер позиции, в которой расположена первая ошибочная закрывающая скобка, или, если закрывающих скобок не хватает, значение −1.
4. Вводится строка, состоящая из слов, разделенных подчеркиваниями (одним или несколькими). Длина строки может быть разной. Определить и вывести количество слов в строке.
ВАРИАНТ № А42/Б4
1. Дано четырехзначное целое ненулевое положительное число N (N>0). Проверить истинность высказывания: "Все цифры данного числа различны".

4. Вводится строка, состоящая из слов (разделенных знаком равенства - '='), содержащая, по крайней мере, один символ '='. Длина строки может быть разной. Вывести подстроку, расположенную между вторым и третьим знаком '=' исходной строки. Если строка содержит менее 3-х символов '=', то вывести всю строку.

ВАРИАНТ № А44/Б2
1. Даны три целых ненулевых числа. Найти среднее из этих чисел (т. е. число, расположенное между наименьшим и наибольшим числом).
2. Из пяти целых различных ненулевых положительных и отрицательных чисел найти самое наибольшее число.
3. Вводится строка-предложение. Длина строки может быть разной. Подсчитать количество содержащихся в строке знаков препинания (точка, запятая, двоеточие, точка с запятой, восклицательный знак, вопросительный знак).
4. Вводится строка. В строке могут быть любые символы. Длина строки может быть разной. Подсчитать и вывести количество содержащихся в этой строке цифр, а так же сумму всех этих цифр.
ВАРИАНТ № А45/Б1
1. Дано четырехзначное целое ненулевое положительное число N (N>0). Проверить истинность высказывания: "Все цифры данного числа одинаковые".
2. Дано количество часов, минут и секунд (1 час = 60 минут, 1 минута = 60 секунд). Вычислить и вывести общее количество секунд.
3. Вводится строка. Длина строки может быть разной. Подсчитать и вывести количество содержащихся в ней прописных букв латинского алфавита.
4. Вводится строка, содержащая буквы и цифры. Длина строки может быть разной. Вывести сумму и произведение цифр этой введенной строки. Чтобы избежать целочисленного переполнения при произведении, вычислять это выражение с помощью вещественной переменной и выводить его как вещественное число.
