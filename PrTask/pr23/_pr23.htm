Практическая работа № 23
"Работа с LINQ"

Цель работы:
Получить первичные навыки по работе с технологией LINQ в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать консольный проект и написать линейные программы (задание - ввод данных с консоли - ответ), использующие консольный ввод-вывод и переменные целого типа и/или вещественного типа.
  2. В каждом варианте есть задания на работу с целыми числами (задания 1-2) и на работу с массивами(задания 3-5). При обработке всех заданий варианта использовать только технологию LINQ
  3. ВНИМАНИЕ!!! В заданиях варианта, где необходимо использовать массивы входные (начальные) значения должны вводиться строго в исходном коде. Выходные значения (результат работы) во всех заданиях варианта необходимо выводить на консоль
  4. Минимальный размер массива должен быть не менее 25 элементов.
  5. Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных, в целях проверки корректности написанного кода.
  6. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  7. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...). Для обработки ошибок использовать механизм исключений платформы .NET.
  8. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  9. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.
