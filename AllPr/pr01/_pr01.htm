МДК.01.01 Разработка программных модулей

Практическая работа № 01
"Простые расчеты используя язык C#"

Цель работы:
Получить первичные навыки по работе с консолью, арифметическими операторами и подсистемы ввода/вывода в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать консольный проект и написать линейные программы (вывод задания на консоль-ввод начальных данных в консоли-вывод ответа на консоль), использующие консольный ввод-вывод (функции ReadLine и WriteLine) и переменные целого типа и/или типа с плавающей запятой. Для конвертирования между различными типами данных использовать специальный встроенный в стандартную библиотеку платформы класс Convert.
  2. Все задания своего варианта выполнять в одном решении (solution) Visual Studio в одном проекте.
  3. ВНИМАНИЕ! Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных, в целях проверки корректности написанного кода.
  4. ВНИМАНИЕ! Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии для каждой/нескольких строк кода.
  5. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.
