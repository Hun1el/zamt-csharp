Практическая работа № 15
"Наследование в языке C#"

Цель работы:
Получить первичные навыки по работе с механизмом наследования в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio создать проект и написать линейные программы
  2. Для задания в каждом варианте необходимо разработать базовый и производный класс, используя механизм наследования
  3. Во всех заданиях реализовать функцию/метод, получающую и возвращающую объекты базового класса
  4. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, количество цифр в вводимом значении и т.д.). Вывод сообщений об ошибках выводить на консоль.
  5. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  6. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...).
  7. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  8. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

  • два (2) правильно выполненных задания 01 и 02 на оценку "4" (хорошо)
  • три (3) правильно выполненных задания 01, 02 и 03 на оценку "5" (отлично)

Задания для практической работы по вариантам:

Задание 01
Создать базовый класс Саr (машина), характеризуемый торговой маркой (строка), числом цилиндров, мощностью. Определить методы переназначения и изменения мощности. Создать производный класс Lorry (грузовик), характеризуемый грузоподъемностью кузова. Определить функции переназначения марки и изменения грузоподъемности.
Задание 02
Создать класс Liquid (жидкость), имеющий поля названия и плотности. Определить методы переназначения и изменения плотности. Создать производный класс Alcohol (спирт), имеющий крепость. Определить методы переназначения и изменения крепости.
Задание 03
Создать класс Man (человек), с полями: имя, фамилия, возраст, пол и вес. Определить методы переназначения имени, изменения возраста и изменения веса. Создать производный класс Student, имеющий поля года обучения и специальность обучения. Определить методы переназначения и увеличения года обучения.
