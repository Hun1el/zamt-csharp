Практическая работа № 04
"Строки в C#"

Цель работы:
Получить первичные навыки по работе со строками в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать консольный проект и написать линейные программы (задание - ввод данных с консоли - ответ), использующие консольный ввод-вывод и переменные целого типа и/или вещественного типа.
  2. В каждом варианте есть задания на работу с целыми числами или массивами (задания 1-2) и на работу со строками (задания 3-5).
  3. ВНИМАНИЕ!!! Во всех заданиях варианта входные (начальные) значения должны вводиться строго с консоли. Выходные значения (результат работы) во всех заданиях варианта необходимо выводить на консоль
  4. Минимальный размер строк(и) (массив символов) - не менее 40 символов.
  5. Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных, в целях проверки корректности написанного кода.
  6. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  7. ВНИМАНИЕ!!! В заданиях, где требуется работа с массивом(и) - обязательно вводить размер массива(ов) и все значения элементов массива(ов) с консоли. Обязательно предусмотреть возможность ручного (с консоли) и автоматического (рандомного) заполнения массива по указанию пользователя.
  8. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...).
  9. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  10. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

Вид учебного занятия:

  • четыре (4) любые правильно выполненные задания своего варианта на оценку "4" (хорошо)

ВАРИАНТ № А1/Б26
1. Дан целочисленный массив, состоящий из N элементов (N > 0). Поменять местами минимальный и максимальный элемент в массиве. Вывести вначале исходный массив, а строкой ниже полученный массив после замены.

3. Вводится строка, состоящая из цифр и символов английского алфавита. Длина строки может быть разной. Вывести на экран произведение всех нечетных цифр этого числа.
4. Вводится строка, содержащая латинские буквы и круглые скобки. Длина строки может быть разной. Если скобки расставлены правильно (т. е. каждой открывающей скобки соответствует одна закрывающая скобка), то вывести число 0. В противном случае вывести или номер позиции, в которой расположена первая ошибочная закрывающая скобка, или, если закрывающих скобок не хватает, значение −1.
5. Вводится строка, состоящая из слов, разделенных подчеркиваниями (одним или несколькими). Длина строки может быть разной. Строка содержит цифры и строчные латинские буквы. Если буквы в строке упорядочены по алфавиту, то вывести 0; в противном случае вывести номер первого символа строки, нарушающего алфавитный порядок.
