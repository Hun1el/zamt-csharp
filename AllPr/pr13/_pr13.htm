Практическая работа № 13
"Перегрузка методов и управление доступом в С#"

Цель работы:
Получить первичные навыки по работе с перегрузкой методов и управлением доступа в языке программирования С#

Задание:

  • Используя интегрированную среду разработки (IDE) Microsoft Visual Studio требуется разработать простой класс для хранения и обработки данных в соответствии с номером Вашего варианта. Для разработки ПР использовать консольный тип проекта.
  • В каждом проекте должен быть свой реализованный класс (один или несколько) и главная функция Main(), из которой этот/эти класс(ы) необходимо вызывать! Одному заданию соответствует один класс.
  • Имя класса(ов) в проекте обязательно должно соответствовать номеру задания из Вашего варианта (taskХХ, где ХХ - номер Вашего задания из варианта)! Все классы должны быть в разных файлах одного проекта! Для выбора имени namespace использовать номер Вашего варианта (например, namespace var18). Задание, где требуется реализовать функцию выполнять в виде метода класса
  • В каждом реализованном классе использовать разграничение доступа (открытый и закрытый) к объектам класса используя ключевые слова private и public.
  • Для задания с выполнением по реализации метода, перегрузить его максимальное кличество раз. Для передачи в метод значений использовать именованные аргументы
  • Проверить работу для статических и динамических экземпляров класса.
  • В каждом варианте есть задания на работу с числами и на работу со строками. В идеале, рекомендуется сделать все задания.
  • Обязательно предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...) используя механизм обработки исключительных ситуаций!!!!!
  • Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале.
  • Материалы проекта сохранить на сервере в своей папке и в личном облачном хранилище.

Продолжительность работы:
2 академических часа

Критерии оценивания выполненной работы:

  • три (3) любых правильно выполненных задания своего варианта на оценку "3" (удовлетворительно)
  • четыре (4) любых правильно выполненных задания своего варианта на оценку "4" (хорошо)
  • пять (5) любых правильно выполненных задания своего варианта на оценку "5" (отлично)
