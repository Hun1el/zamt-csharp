Практическая работа № 17
"Работа с системным реестром ОС MS Windows"

Цель работы:
Получить первичные навыки по работе с системным реестром в ОС MS Windows в языке программирования С#

Общее задание:
1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio создать проект и написать линейные программы используя хранилище входных параметров системный реестр в ОС MS Windows.
2. Входные параметры и результаты работы заданий необходимо сохранять в системном реестре ОС MS Windows в разделе HKEY_CURRENT_USER в созданном подкаталоге PR_17. В подкаталоге создать еще подкаталог taskXX, где XX - номер задания. Все задания из своего варианта выполнять в новом проекте, но в одном решении (solution) Visual Studio. После защиты ПР обязательно удалить подкаталог PR_17 из раздела HKEY_CURRENT_USER. Для входных значений использовать осмысленные имена, исходя из логики входного параметра. Результат вычисления задания необходимо записать в переменную resultN, где N - номер выпролняемого задания. Имена переменных записанные в системном реестре должны быть на иностранном языке.
3. Обязательно предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...) используя механизм обработки исключительных ситуаций!
4. ВНИМАНИЕ! Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio (никаких "лесенок" из кода быть не должно) и содержать комментарии для каждой/нескольких строк кода.
5. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.
