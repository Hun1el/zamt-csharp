МДК.01.01 Системное программирование

Практическая работа № 02
"Циклы и условия в языке C#.
Обработка исключительных ситуаций"

Цель работы:
Получить первичные навыки по работе с циклами, условиями и механизмом обработок ошибок в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать консольный проект и написать линейные программы (задание - ввод данных с консоли - ответ), использующие консольный ввод-вывод и переменные целого типа и/или вещественного типа.
  2. Все задания варианта делать в одном проекте Microsoft Visual Studio! Программный код должен последовательно решать первую задачу (ввод данных - решение - вывод результата), затем вторую и после последующие задания. Во всех заданиях варианта дается работа с целочисленными значениями. В пятом задании во всех вариантах обязательно использовать оператор выбора языка C# SWITCH-CASE.
  3. ВНИМАНИЕ!!! Во всех заданиях варианта входные (начальные) значения должны вводиться строго с консоли. Выходные значения (результат работы) во всех заданиях варианта необходимо выводить на консоль
  4. Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных, в целях проверки корректности написанного кода.
  5. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  6. Предусмотреть контроль ввода начальных значений на корректность/правильность используя механизм обработки ошибок платформы - исключений (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...).
  7. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  8. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

2. Даны ненулевые числа x, y. Проверить истинность высказывания: «Точка с координатами (x, y) лежит во второй координатной четверти».
3. Проверить истинность высказывания: "Все цифры данного целого положительного пятизначного числа введенного с клавиатуры различны".
4. Дано целое положительное пятизначное число N (N > 0). Найти разницу между суммой всех его цифр и произведением четных цифр.
5. Дано целое число N в диапазоне от -9000 до 9000. Вывести строку-описание данного числа, например: "положительное четырехзначное число" или "отрицательное двузначное".

ВАРИАНТ № А20/Б6
1. Даны два целых положительных числа A и B (A < B). Найти произведение всех нечетных чисел расположенных между этими числами A и B.
2. Проверить истинность высказывания: "Данное целое положительное число является четным трехзначным числом".

4. Даны два целых положительных числа A и B (число A меньше числа B). Вывести все нечетные числа, расположенные между данными числами (не включая сами эти числа), в порядке их убывания, а также количество этих чисел и их сумму.
5. Дано целое положительное число K в диапазоне от 1 до 7. Вывести строку - название дня недели, соответствующее данному числу (1 - «понедельник», 2 - «вторник», ..., 7 - «воскресенье»). Если K не лежит в диапазоне от 1 до 7, то вывести строку с сообщением "Ошибка"/"Error".
ВАРИАНТ № А21/Б5
1. Дан размер некоторого файла в байтах (целочисленное положительное ненулевое шестизначное число). Используя операцию деления нацело, найти количество полных килобайтов и мегабайтов, которые занимает данный файл (1 килобайт = 1024 байта, 1 мегабайт - 1024 килобайт).
2. Проверить истинность высказывания: "Данные целые числа X и Y являются координатами точки, лежащей во второй координатной четверти".
3. Дано целое положительное четырехзначное ненулевое число N (N>0). Проверить истинность высказывания: «Данное число N читается одинаково слева направо и справа налево».
4. Дано целое число N (N > 0). Вывести произведение 1*2*3*...*N. Чтобы избежать целочисленного переполнения, вычислять это произведение с помощью вещественной переменной и выводить его на экран как вещественное число. Пример для числа 5 - 1*2*3*4*5=120.
5. Арифметические действия над числами пронумерованы следующим образом: 1 - сложение, 2 - вычитание, 3 - умножение, 4 - деление. Дан номер действия N (целое число в диапазоне от 1 до 4) и вещественные числа A и B (B не равно 0). Выполнить над числами указанное действие и вывести результат.

4. Дано вещественное число A и целое число N (N > 0). Вывести значение результата A в степени N: AN = A*A*...*A (число A перемножается N раз).
5. Дано целое положительное число K в диапазоне от 1 до 7. Вывести строку - название дня недели, соответствующее данному числу (1 - «понедельник», 2 - «вторник», ..., 7 - «воскресенье»). Если K не лежит в диапазоне от 1 до 7, то вывести строку с сообщением "Ошибка"/"Error".
