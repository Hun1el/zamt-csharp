Практическая работа № 07
"Файловый ввод/вывод в языке С#"

Цель работы:
Получить первичные навыки по работе с файловым вводом/выводом в языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать проект и написать программы используя встроенные функции в библиотеку языка С# и консольный ввод-вывод, переменные целого типа и/или типа с плавающей запятой.
  2. Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных (в текстовых файлах), в целях проверки корректности/правильности написанного кода.
  3. Все входные и выходжные данные должны быть только в текстовых файлах. Для каждого задания в варианте необходимо создать два текстовых файла. Первый файл - это входные значения по заданию (целое число - одно или несколько, массив чисел или строка). Название первого файла - inputN, где N - это номер задания. Второй файл - это выходные данные, результат работы одного задания (целое число - одно или несколько, массив чисел или строка). Название второго файла - outputN, где N - это номер задания. Работу выполнять в одном проекте VS.
  4. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  5. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  6. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

Вид учебного занятия:

Продолжительность учебного занятия:

  • два (2) любых правильно выполненных задания своего варианта на оценку "3" (удовлетворительно)
  • три (3) любых правильно выполненных задания своего варианта на оценку "4" (хорошо)
  • четыре (4) любых правильно выполненных задания своего варианта на оценку "5" (отлично)
