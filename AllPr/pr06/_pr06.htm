Практическая работа № 06
"Исследование и защита приложения на языке C#"

Цель работы:
Получить первичные навыки при работе в исследовании и защите приложения написанном на языке программирования С#

Общее задание:

  1. Используя интегрированную среду разработки (IDE) Microsoft Visual Studio или JetBrains Rider создать консольный проект и написать линейные программы (задание - ввод данных с консоли - ответ), использующие консольный ввод-вывод и переменные целого типа и/или вещественного типа.
  2. ВНИМАНИЕ!!! Во всех заданиях варианта входные (начальные) значения должны вводиться строго с консоли. Выходные значения (результат работы) во всех заданиях варианта необходимо выводить на консоль
  3. Минимальный размер строк(и) (массив символов) - не менее 40 символов. Минимальный размер массива чисел - не менее 25 элементов.
  4. Проверить весь код ПР используя статический анализатор PVS-Studio. Выполнить упаковку исполняемого файла
  5. Сдать проект, показав построенную цепочку команд в отладчике, убрав при этом все "точки остановки" отладчика, кроме последней. Во время защиты (сдачи) практической работы преподаватель может изменить начальный набор входных данных, в целях проверки корректности написанного кода.
  6. Код на языке C# должен быть отформатирован по стилю оформления редактора кода в MS Visual Studio или JetBrains Rider (никаких "лесенок" из кода быть не должно) и содержать комментарии в исходном коде при решении алгоритма.
  7. ВНИМАНИЕ!!! В заданиях, где требуется работа с массивом(и) - обязательно вводить размер массива(ов) и все значения элементов массива(ов) с консоли. Обязательно предусмотреть возможность ручного (с консоли) заполнения и автоматического (рандомного) заполнения массива по указанию пользователя.
  8. Предусмотреть контроль ввода начальных значений на корректность/правильность (например, выход за диапозон указанных чисел/значений, деление на 0, ввод только положительных/отрицательных значений, ...).
  9. Номер варианта для выполнения практической работы соответствует номеру студента/обучающегося по списку в учебном журнале группы.
  10. Материалы проекта обязательно сохранить на сервере в своей папке и в личном облачном хранилище.

Вид учебного занятия:
практическое занятие (аудиторная самостоятельная защита работ под руководством преподавателя)

Продолжительность учебного занятия:
2 академических часа

примеры проектов Visual Studio

  • четыре (4) любые правильно выполненные задания своего варианта на оценку "4" (хорошо)

ВАРИАНТ № А1/Б26
1. Даны три целых числа: A, B, C. Проверить истинность высказывания: «Ровно одно из чисел A, B, C положительное».
2. В вещественном массиве хранятся сведения о количестве осадков, выпавших за каждый день месяца N (в месяце должно быть 30 дней). Определить, сколько осадков выпадало в среднем за один день в первую, вторую и третью декады этого месяца (декада состоит из 10 дней). Предоставить возможность пользователю реализовать заполнение элементов массива случайными (рандомными) числами.
3. Дан целочисленный массив, состоящий из N элементов (N > 0). Найти и вывести количество элементов, расположенных после самого последнего максимального элемента.
4. Вводится строка, изображающая целочисленное арифметическое выражение вида «цифра_цифра_цифра_цифра», где на месте знака операции «_» находится символ «+» или «-», а на месте "цифра" находится одна из цифр (от 1 до 9). Например, «4+7-2+5». Вывести значение данного выражения (как целое число).
5. Вводится строка, состоящая из слов, разделенных подчеркиваниями (одним или несколькими). Длина строки может быть разной. Определить и вывести количество слов, у которых второй и последний символ одинаковые.
